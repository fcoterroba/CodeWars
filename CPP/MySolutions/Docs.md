# C++ docs

## Create variables

As many other languages, to create a variable you need to follow `typeOfVariable nameOfVariable [= optional result];`. Examples:
- `string MyName = "Francisco";`
- `string YourAge;`
- `int MyAge = 25;`

## If/Else

The if/else statements are classical.
```cpp
if (YourAge < 18) {
  return "Under age.";
} else {
  return "Welcome, adult.";
}
```
