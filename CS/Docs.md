# What is `static`?

Static is a function's keyword used to make a static function. That means that you **DON'T** need to create an object to call that function

# Functions

In C#, functions need the type of value returned, for examples:

```cs
public static string myString(){
    // ...
}

public static bool myBooleans(){
    // ...
}

public static int myInteger(){
    // ...
}
```
