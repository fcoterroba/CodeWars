# Basic info

To execute a cpp file, you must compile it with `g++ nameOfFile.cpp`, then execute the resultant file like `./a.exe`

## Main function

CPP use a main function as many other languages. So, to run something in cpp you must create a main function called `int main()`

## Printing

For print text, you need first use the **std** namespace like `using namespace std;` and before include the **iostream** with `#include <iostream>`. Then print whatever with `cout<<"You print here";`

## Functions

The parameters of the functions must include the expected type of parameter at the beginning and at the end like `int32_t double_integer(int32_t n)`

## Useful functions

### stoi (String TO Integer)

With `stoi(variable)` you can convert a string into a integer. (stoi is abreviation of "**S**tring **TO** **I**nteger")
