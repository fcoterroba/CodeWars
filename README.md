# CodeWars

Repo to save my answers to the daily CodeWars katas I do

Every weekday, I try to complete a CodeWars kata. From Monday to Wednesday, I do one focusing on "Fundamentals" for the languages ​​I'm learning: C++, C#, and Rust. On Thursdays and Fridays, I do one focusing on "Rank Up" for languages ​​I've already learned: PHP and Python.
