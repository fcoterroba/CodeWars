# CodeWars

This repository documents my weekly 5 katas aimed at learning the languages I consider essential for the development of the **metaverse**. Each week, I'll focus on a different language to improve my skills and prepare to contribute to the metaverse.

## Weekly Kata Organization

Each day of the week is dedicated to a different language:

- **Monday**: **C#**
- **Tuesday**: [**Python**](https://github.com/fcoterroba/CodeWars/tree/main/Python)
- **Wednesday**: **C++** 
- **Thursday**: **PHP**
- **Friday**: **Rust** 

## How to Contribute

1. **Find the Katas**: Inside each folder corresponding to a language (C#, Python, C++, PHP, Rust), you'll find the weekly katas. Each kata contains a description and its solution.
2. **Submit Your Solutions**: If you're a programmer and want to share your solution, feel free to open a *pull request* with your proposed solution.
3. **Learn Together**: This repository is open to anyone who wants to learn and contribute, so don't hesitate to join in!

## How to Get Started

1. Search if your kata is included yet.
2. Submit your solution through a *pull request*.
