# CodeWars

My public repo to save the solutions I create for the Codewars' katas I play. Feel free to add yours

## Organization

Every weekday, I tackle a kata in one of these 5 languages:  
- Monday 🐍 [**Python**](https://github.com/fcoterroba/CodeWars/tree/main/Python) on Rank up focus
- Tuesday 🐘 [**PHP**](https://github.com/fcoterroba/CodeWars/tree/main/PHP) on Rank up focus
- Wednesday 🦀 [**Rust**](https://github.com/fcoterroba/CodeWars/tree/main/Rust/) on Fundamental's focus
- Thursday ⚙️ [**C++**](https://github.com/fcoterroba/CodeWars/tree/main/CPP) on Fundamental's focus  
- Friday 🎮 [**C#**](https://github.com/fcoterroba/CodeWars/tree/main/CS) on Fundamental's focus

You’re welcome to contribute **your own unique solutions** for the katas in any of these languages! Just make sure your solution is unique comparing with the ones already uploaded.

## 📁 Repository Structure  
To keep things organized, the repo is structured as follows:  

```
CodeWars/
├── python/
│   ├── kata-name/
│   │   ├── MySolution.py
│   │   └── <YourName>Solution.py
├── php/
│   ├── kata-name/
│   │   ├── MySolution.php
│   │   └── <YourName>Solution.php
├── rust/
│   ├── kata-name/
│   │   ├── MySolution.rs
│   │   └── <YourName>Solution.rs
├── cpp/
│   ├── kata-name/
│   │   ├── MySolution.cpp
│   │   └── <YourName>Solution.cpp
├── csharp/
│   ├── kata-name/
│   │   ├── MySolution.cs
│   │   └── <YourName>Solution.cs
└── README.md
```
