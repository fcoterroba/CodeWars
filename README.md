Welcome to **CodeWars**, where I commit to uploading my solutions to coding katas every weekday. This repository is all about continuous improvement and tackling coding challenges across various programming languages.

## 🚀 Goal
My goal is to improve my coding skills by solving one kata each day in the language assigned for that day. This journey is all about consistency and learning something new every day.

## 📂 Repository Structure
Each folder is organized by language and contains:
- The solution to the kata.
- Maybe some notes or thoughts about the language.

## 🎯 What is a "Kata"?
In programming, a kata is a challenge designed to practice coding skills. The katas in this repository come from [CodeWars](https://www.codewars.com/), a platform where you can train by solving coding problems.
