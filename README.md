# CodeWars
Welcome! This is a space for programmers to sharpen their skills, share knowledge, and explore different approaches to coding challenges. 🚀  

## 🔍 What is this about?  
Every weekday, I tackle a kata in one of these 5 languages:  
- Monday 🐍 [**Python**](https://github.com/fcoterroba/CodeWars/tree/main/Python)
- Tuesday 🐘 [**PHP**](https://github.com/fcoterroba/CodeWars/tree/main/PHP)
- Wednesday 🦀 [**Rust**](https://github.com/fcoterroba/CodeWars/tree/main/Rust/)
- Thursday ⚙️ [**C++**](https://github.com/fcoterroba/CodeWars/tree/main/CPP)  
- Friday 🎮 [**C#**](https://github.com/fcoterroba/CodeWars/tree/main/CS)

You’re welcome to contribute **your own unique solutions** for the katas in any of these languages! Just make sure your solution differs from the ones already uploaded.

## 📁 Repository Structure  
To keep things organized, the repo is structured as follows:  

```
CodeWars/
├── python/
│   ├── kata-name/
│   │   ├── MySoltion.py
│   │   └── <YourName>Solution.py
├── php/
│   ├── kata-name/
│   │   ├── my-solution.php
│   │   └── <YourName>Solution.php
├── rust/
│   ├── kata-name/
│   │   ├── my-solution.rs
│   │   └── <YourName>Solution.rs
├── cpp/
│   ├── kata-name/
│   │   ├── my-solution.cpp
│   │   └── <YourName>Solution.cpp
├── csharp/
│   ├── kata-name/
│   │   ├── my-solution.cs
│   │   └── <YourName>Solution.cs
└── README.md
```

- Each **language folder** contains subfolders named after the katas.  
- Inside each kata folder, you’ll find the solutions with filenames like `MySolution` for my solution or upload yours using `<YourName>Solution`.

## 📝 How to Contribute  
We’d love to see your unique approaches! Here’s how you can contribute:  

1. **Fork the repository** and clone it to your machine.  
2. Navigate to the correct language and kata folder.  
3. Add your solution with a filename like `<YourName>Solution`.  
4. Commit and push your changes.  
5. Open a **pull request** explaining why your solution is unique or interesting!
